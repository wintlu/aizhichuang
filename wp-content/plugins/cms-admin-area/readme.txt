Simple plugin adds CMS-like features:  user-friendly horizontal menu and  dashboard metaboxes.

== Description ==

This simple plugin provides a nice CMS admin interface:

1. adds user-friendly horizontal menu and  dashboard metaboxes
2. completely removes blogging plarform widgets

It gives your clients a better experience of their new website.

Support for translation!

- Polish translation
- Dutch translation by Jos Wolbers


== Installation ==

1. Upload `cms-admin-area` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done!
